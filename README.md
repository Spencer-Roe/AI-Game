# Artificial Evidence
Our game allows you to solve AI-generated murder mysteries by talking to different characters in order to crack the case. On startup it creates a completely unique scenario. It creates a victim assigns the killer and creates alibis and clues to get from the characters.  
[![Watch the video](https://youtu.be/DYX_jkU-WXE?si=Zjz09NVyuUIaS0n-)](https://youtube.com)



To play this game you will ned to create a textFile with the name .env in the same file that has the Library Assets ect. 
in this text file you will need to put in a API ket to use with the format :
OPENAI_API_KEY=sk-proj-xn ...
